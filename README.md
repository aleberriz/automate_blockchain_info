# automate_blockchain_info

Early automation project (2021) to extract, aggregate, and export event data from a blockchain info source — in this case, WIPP (a blockchain-based waste management registry). Outputs structured CSVs for downstream analysis.

> Preserved for portfolio reference as an early Python automation example. Not actively maintained.

## What this demonstrates

- Web/API data extraction and automation using Python
- Data cleaning and aggregation with Pandas
- Output to structured CSV for analysis

## Contents

| File | Description |
|------|-------------|
| `WIPP_bookeeping_automation.ipynb` | Main notebook: extraction, transformation, export |
| `WIPP_events_allAvailableInfo.csv` | Full event data export |
| `WIPP_events_crosssection.csv` | Cross-section of events |
| `WIPP_events_summary.csv` | Aggregated summary |

## Stack

- Python, Pandas, Jupyter Notebook
